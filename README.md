**🥗 Personalized Diet & Nutrition Plan Generator**

A machine learning based web application that generates personalized diet plans based on individual health parameters such as age, weight, height, activity level, dietary preference, and health goals.

This system helps users maintain healthier lifestyles and manage lifestyle diseases like obesity, diabetes, and cardiovascular conditions by recommending balanced daily meals.

**📌 Project Overview**

Modern lifestyles often lead to unhealthy eating habits and reduced physical activity, resulting in lifestyle diseases such as obesity, diabetes, and heart related disorders.

The Personalized Diet & Nutrition Plan Generator analyzes user health data and generates a customized diet plan that meets the individual's daily nutritional requirements.

The system calculates important health metrics such as:
Body Mass Index (BMI)
Basal Metabolic Rate (BMR)
Total Daily Energy Expenditure (TDEE)

Using these calculations, the system determines the user's daily calorie requirement and generates balanced meals for:
Breakfast
Lunch
Dinner

The application is built with Python and Streamlit, providing an interactive interface for users to generate diet plans instantly.

**🚀 Features :**
✔ Personalized diet plan generation
✔ BMI calculation and health category detection
✔ BMR and TDEE calculation
✔ Goal based calorie adjustment (Weight Loss / Maintain / Weight Gain)
✔ Support for Vegetarian, Non-Vegetarian, and Mixed diets
✔ Food intolerance filtering
✔ Balanced meal generation for Breakfast, Lunch, and Dinner
✔ Portion size and calorie display for each food item
✔ Interactive web interface using Streamlit

**🧠 Machine Learning Approach**
The system uses machine learning techniques to classify user dietary requirements based on health parameters.

Algorithms Used
Logistic Regression
Random Forest Classifier

These models help classify the user into appropriate diet categories such as:
Weight Loss Diet
Maintenance Diet

Weight Gain Diet

**⚙️ System Workflow**
User Input
     ↓

     ↓
Daily Calorie Requirement
     ↓
Diet Preference Filtering
     ↓
Food Intolerance Filtering
     ↓
Meal Dataset Selection
     ↓
Smart Meal Generation Algorithm
     ↓
Personalized Diet Plan

**🗂 Dataset**
The system uses an Indian Meal Nutrition Dataset that contains nutritional values for commonly consumed foods.

Dataset attributes include:
Food Name
Meal Type (Breakfast / Lunch / Dinner)
Diet Type (Veg / NonVeg)
Calories
Protein
Carbohydrates
Fat

This dataset enables the system to generate realistic meal combinations based on nutritional requirements.
